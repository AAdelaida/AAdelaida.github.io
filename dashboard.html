Dashboard (Open Data)

Sreamlit, Matplotlib, Pandas, Seaborn

Código Ver en Streamlit

This is a dashboard app created in Python using Streamlit.
Streamlit is an open-source Python library that makes it easy to create and share beautiful, custom web apps for machine learning and data science.
The data is downloaded from the official open data site of the Government of Mexico https: //datos.gob.mx/, in which data with the necessary technical and legal characteristics are published so that anyone, anywhere and at any time, can freely use and distribute them.
